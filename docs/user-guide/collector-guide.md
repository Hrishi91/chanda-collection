# Collector guide / যারা চাঁদা তুলবে তাদের জন্য

## Install (একবারই)

1. Chrome-এ app-এর link খোলো (Hrishi পাঠাবে)।
2. Menu (⋮) → **Add to Home screen / হোম স্ক্রিনে যোগ করুন** → Add।
3. Home screen-এর 🙏 icon থেকে খোলো — app-এর মতোই চলবে, internet
   ছাড়াও।

## নাম লেখানো ও লগইন

1. **নতুন? নাম লেখাও (register)** — নিজের নাম, একটা username
   (ইংরেজি ছোট হাতের), ফোন নম্বর আর পাসওয়ার্ড দাও।
2. Hrishi (admin) approve করার পর লগইন করতে পারবে — approve হয়েছে
   কিনা Hrishi-কে জিজ্ঞেস করো।
3. **পাসওয়ার্ড ভুলে গেলে** Hrishi-কে বলো — সে সাময়িক পাসওয়ার্ড
   দেবে; সেটা দিয়ে ঢুকলেই app নতুন পাসওয়ার্ড বসাতে বলবে।
4. লগইন একবারই লাগে — তারপর internet ছাড়াও app খুলবে ও এন্ট্রি হবে।

## এন্ট্রি করা

- **দোকান / ব্যক্তি / সদস্য** — app এক-এক করে প্রশ্ন করবে। 🎤 চেপে
  বলো বা টাইপ করো। টাকা কথায় বললেও হবে: "পাঁচশো" → ₹500।
- সব শেষে **"সব ঠিক আছে?"** — ভুল থাকলে ✏️ চেপে সেই ঘরটা ঠিক করো,
  তারপর **সেভ করো**।
- **পরপর দোকান (bulk)**: এক রাস্তার সব দোকান পরপর তুলতে — রাস্তা
  একবার বাছলেই পরের দোকানে আবার জিজ্ঞেস করবে না।
- **টাকা জমা (কিস্তি)**: খাতা 📒 → নাম খোঁজো → দোকান/মানুষটা খোলো →
  💰 টাকা জমা। কে কত বাকি রেখেছে তাও খাতায় লাল করে দেখা যায়।
- **রোড / টোটো / বাস**: দিনে যতবার খুশি এন্ট্রি। বাসের বেলায় নাম +
  নম্বর লাগবে। Collection চালানোর নিজের খরচ থাকলে entry-র পরে
  "কালেকশন খরচ"-এ লেখো।
- **🧾 খরচ (পুজোর খরচ)**: শুধু **ক্যাশিয়ার/admin**-এর হোমে দেখাবে —
  পুজোর টাকা তো তাদের হাতেই থাকে, তারাই খরচ লেখে।

## Sync — খুব জরুরি ⚠️

- উপরে ডানদিকে **✅** মানে সব জমা পড়েছে; **⏳ সংখ্যা** মানে অত-গুলো
  এন্ট্রি এখনো পাঠানো হয়নি।
- Network পেলেই app নিজে পাঠিয়ে দেয়; ⏳ দেখলে network-এ এসে badge-এ
  চাপ দাও।
- **⏳ থাকা অবস্থায় app delete/clear কোরো না** — ওই এন্ট্রিগুলো শুধু
  তোমার ফোনেই আছে।

## টাকা কীভাবে দিল — নগদ / UPI / দুটোই

- প্রতিটা টাকার এন্ট্রিতে app জিজ্ঞেস করবে **কীভাবে দিল** — নগদ, UPI,
  নাকি দুটোই। "দুটোই" বাছলে নগদ কত + UPI কত আলাদা করে লিখবে।
- UPI যার নম্বরেই আসুক, সেটা **তার হাতের টাকা** বলে খাতায় থাকবে
  যতক্ষণ না ক্যাশিয়ারকে জমা দেওয়া হচ্ছে।

## দিনশেষে টাকা জমা (handover) 🤝

1. হোমে **🤝 জমা দিলাম** → কোন ক্যাশিয়ারকে দিলে → নগদ/UPI কত → সেভ।
2. ক্যাশিয়ার তার ফোনে **✅ জমা নেওয়া confirm**-এ গিয়ে **জমা নিলাম**
   চাপলে হিসাব মিলে যাবে।
3. রিপোর্টের **"কার হাতে কত টাকা"** টেবিলে সবাই দেখতে পাবে কার কাছে
   এখনো কত আছে — তাই দিনশেষে জমা দিয়ে হিসাব শূন্য করে দাও।

## রিপোর্ট

- 📊 রিপোর্ট → উপরে **📱 আমার হিসাব** (সবাই নিজেরটা দেখে): আমি কত
  তুলেছি, ক্যাশিয়ারকে কত জমা দিয়েছি, আমার কাছে কত জমা পড়েছে, **আমার
  হাতে এখন কত**, নগদ/UPI, রোড/টোটো/বাস, আর আমার নিজের খরচ।
  - **আমার হাতে** = তুললাম + আমার কাছে জমা − আমি জমা দিলাম − আমার খরচ।
  - ক্যাশিয়ার নিজের খরচ ও হাতের টাকা এখানেই দেখবে।
- নিচে **☁️ কেন্দ্রীয় রিপোর্ট** — এখানে সেই report-গুলোই দেখাবে যেগুলো
  admin তোমাকে দেখার permission দিয়েছে।
- ক্যাশিয়ার হলে **"কার হাতে কত টাকা"** এমনিই দেখতে পাবে।
- অন্য কোনো report (মোট হিসাব, বাকির তালিকা, খরচ, ইত্যাদি) দরকার হলে
  admin-কে বলো — সে permission দিলে তোমার এখানে চলে আসবে।
- সব report শুধু **দেখার** — এখানে কিছু বদলানো যায় না।

## Admin-এর জন্য (report permission)

সেটিংস → 👑 Admin প্যানেল → প্রতিটা approved সদস্যের নিচে **📊 রিপোর্ট
permission**-এর chip আছে। যে report-টা ওকে দেখাতে চাও সেটায় চাপো (আবার
চাপলে সরে যাবে)। Admin নিজে সব report দেখে; ক্যাশিয়ার "কার হাতে কত"
আপনিই পায়।
