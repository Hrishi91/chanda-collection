# Collector guide / যারা চাঁদা তুলবে তাদের জন্য

## Install (একবারই)

1. Chrome-এ app-এর link খোলো (Hrishi পাঠাবে)।
2. Menu (⋮) → **Add to Home screen / হোম স্ক্রিনে যোগ করুন** → Add।
3. Home screen-এর 🙏 icon থেকে খোলো — app-এর মতোই চলবে, internet
   ছাড়াও।
4. প্রথমবার নিজের **নাম** দাও — প্রতিটা এন্ট্রিতে এই নাম বসবে।
5. সেটিংস-এ Sync URL + secret বসাও (Hrishi দেবে)।

## এন্ট্রি করা

- **দোকান / ব্যক্তি / সদস্য** — app এক-এক করে প্রশ্ন করবে। 🎤 চেপে
  বলো বা টাইপ করো। টাকা কথায় বললেও হবে: "পাঁচশো" → ₹500।
- সব শেষে **"সব ঠিক আছে?"** — ভুল থাকলে ✏️ চেপে সেই ঘরটা ঠিক করো,
  তারপর **সেভ করো**।
- **পরপর দোকান (bulk)**: এক রাস্তার সব দোকান পরপর তুলতে — রাস্তা
  একবার বাছলেই পরের দোকানে আবার জিজ্ঞেস করবে না।
- **টাকা জমা (কিস্তি)**: খাতা 📒 → নাম খোঁজো → দোকান/মানুষটা খোলো →
  💰 টাকা জমা। কে কত বাকি রেখেছে তাও খাতায় লাল করে দেখা যায়।
- **রোড / টোটো / বাস**: দিনে যতবার খুশি এন্ট্রি। বাসের বেলায় নাম +
  নম্বর লাগবে। Collection চালানোর খরচ থাকলে "কালেকশন খরচ"-এ লেখো।

## Sync — খুব জরুরি ⚠️

- উপরে ডানদিকে **✅** মানে সব জমা পড়েছে; **⏳ সংখ্যা** মানে অত-গুলো
  এন্ট্রি এখনো পাঠানো হয়নি।
- Network পেলেই app নিজে পাঠিয়ে দেয়; ⏳ দেখলে network-এ এসে badge-এ
  চাপ দাও।
- **⏳ থাকা অবস্থায় app delete/clear কোরো না** — ওই এন্ট্রিগুলো শুধু
  তোমার ফোনেই আছে।

## রিপোর্ট

- 📊 রিপোর্ট → নিজের ফোনের হিসাব।
- **☁️ কেন্দ্রীয় রিপোর্ট** → সবার মিলিয়ে মোট আদায়, খরচ, হাতে আছে,
  কে কত তুলল — এটাই ফাইনাল হিসাব।
